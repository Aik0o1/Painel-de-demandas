# Gerenciador de Demandas

Sistema de Gestão de Demandas desenvolvido com React.

## Tecnologias Utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como rodar o projeto

1. Instale as dependências:
```sh
npm install
```

2. Inicie o servidor de desenvolvimento:
```sh
npm run dev
```
